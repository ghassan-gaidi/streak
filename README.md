# Streak → Dominion

**This project has been consolidated into Dominion.**

Streak was a minimal crypto price API. Dominion is the full product — same price data plus whale tracking, portfolio management, gas tracker, Telegram bot, on-chain tools, and more.

👉 **[Dominion Repository](https://github.com/ghassan-gaidi/crypto-empire)**  
👉 **[Live Dashboard](https://crypto-empire-ten.vercel.app)**  
👉 **[Telegram Bot](https://t.me/Crypt0Emp1reBot)**  

## Why?

Streak and Dominion served the same purpose (crypto market data) at different scopes. Rather than maintaining two overlapping products, all effort goes into Dominion — the more capable, better-tested platform.

## What about the API?

All Streak endpoints are available in Dominion plus 25+ more. The live worker at `crypto-price-api.leo2574.workers.dev` will remain operational for existing integrations but is no longer under active development.
